后台管理系统（Admin 端）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

后台面向运营人员和管理员使用，需要独立登录入口，与前台用户端完全分开。

────────────────────────────────────────
3.1 管理员权限体系
────────────────────────────────────────

角色分级：

  超级管理员（最高权限）：
    · 可以访问全部模块
    · 可以新增/删除/修改其他管理员账号
    · 可以修改平台核心配置

  运营管理员：
    · 用户管理、内容管理、积分管理、订单管理
    · 不可访问系统配置和财务结算模块

  财务管理员：
    · 仅可访问订单管理、收入统计、提现管理
    · 不可访问用户隐私信息

  客服管理员：
    · 仅可访问用户管理（查看为主）、工单/投诉管理、积分手动补发

各角色权限可在「角色权限配置」模块中精细化配置（具体权限项粒度到每个菜单和操作按钮）。

管理员账号管理：
  · 账号列表：账号名 / 角色 / 创建时间 / 最近登录时间 / 状态（启用/禁用）
  · 新增账号 / 修改密码 / 禁用账号
  · 登录需要账号密码 + 短信验证码双重验证


────────────────────────────────────────
3.2 数据概览仪表盘
────────────────────────────────────────

默认登录后首页，展示平台核心运营数据。

今日实时数据卡片（自动刷新）：
  · 今日新注册用户数
  · 今日活跃用户数（产生过操作的用户）
  · 今日 PPT 生成次数（成套+单页合计，分别标注）
  · 今日下载次数
  · 今日新增订单数
  · 今日收入金额（元）
  · 今日积分消耗总量
  · 当前系统任务队列积压数（等待生成的任务数量）

趋势折线图（可选择时间范围：7天/30天/90天/自定义）：
  · 新用户注册趋势
  · 日活用户趋势
  · PPT 生成量趋势
  · 收入趋势
  · 积分消耗趋势

关键比率卡片：
  · 注册到首次生成转化率
  · 生成到下载转化率
  · 免费用户转付费率
  · 付费用户次月留存率
  · 平均每用户生成次数

生成成功率监控：
  · 过去 24 小时生成任务成功率（成功数/总提交数）
  · 失败任务数量及原因分布饼图（AI 调用失败/超时/内容违规等）
  · 平均生成耗时（秒）


────────────────────────────────────────
3.3 用户管理
────────────────────────────────────────

用户列表：
  · 搜索：按手机号 / 用户ID / 昵称精确搜索
  · 筛选：注册时间范围 / 是否付费过 / 账号状态（正常/封禁）/ 注册渠道
  · 排序：按注册时间 / 最近活跃时间 / 总消费金额 / 积分余额
  · 列表字段：用户ID / 昵称 / 手机号（脱敏）/ 注册时间 / 最近登录时间 /
              积分余额 / 累计消费金额 / 生成总次数 / 下载总次数 / 账号状态

用户详情页（点击某用户进入）：
  基本信息：
    · 头像 / 昵称 / 用户ID / 手机号 / 微信openid（脱敏）/ 注册时间 / 最近登录IP

  积分信息：
    · 当前积分余额
    · 累计获得积分（充值 + 注册赠送 + 邀请奖励 分项合计）
    · 累计消耗积分
    · 积分流水明细列表（每条记录：时间/类型/变动积分数/变动后余额/备注）

  消费信息：
    · 历史订单列表
    · 累计消费金额

  生成记录：
    · 该用户历史所有生成记录列表
    · 每条：生成时间 / 类型（成套/单页）/ PPT标题 / 页数 / 消耗积分 /
            生成状态（成功/失败）/ 是否已下载

  操作记录：
    · 登录日志：时间 / IP / 设备类型
    · 关键操作日志：充值/积分变动等操作记录

  管理操作：
    · 手动充值积分（需填写原因，记录操作日志）
    · 手动扣减积分（需填写原因）
    · 封禁账号（需填写封禁原因和封禁时长或永久）
    · 解除封禁
    · 强制退出登录（使该用户所有设备的 token 失效）
    · 发送站内通知消息


────────────────────────────────────────
3.4 PPT 生成记录管理
────────────────────────────────────────

生成任务列表：
  · 搜索：按用户ID / PPT标题 / 任务ID
  · 筛选：生成时间范围 / 类型（成套/单页）/ 状态（成功/失败/生成中/排队中）
  · 列表字段：任务ID / 用户昵称 / PPT标题 / 类型 / 页数 / 风格 /
              提交时间 / 完成时间 / 耗时（秒）/ 消耗积分 / 状态 / 是否已下载

任务详情页：
  · 用户输入的原始内容（主题/正文/上传素材记录）
  · AI 生成的大纲内容
  · 用户最终修改后的大纲
  · 生成结果的页面缩略图预览
  · 失败原因（如失败）
  · 重新触发生成按钮（管理员可手动重试失败任务，不再扣用户积分）

内容合规管理：
  · 对生成内容涉及违规词/政治敏感内容进行标记
  · 标记列表：时间 / 用户 / 触发关键词 / 处理状态
  · 管理员可标记为「已处理」并决定是否退还积分


────────────────────────────────────────
3.5 积分与订单管理
────────────────────────────────────────

【订单管理】

订单列表：
  · 搜索：订单号 / 用户手机号 / 第三方支付流水号
  · 筛选：支付时间范围 / 支付方式（微信/支付宝）/ 订单状态（待支付/已完成/已退款）
  · 列表字段：订单号 / 用户 / 套餐名称 / 金额 / 积分数量 /
              支付方式 / 支付时间 / 状态

订单详情页：
  · 完整订单信息
  · 支付回调记录
  · 退款操作（部分退款/全额退款，需填写退款原因）
  · 退款后自动扣回对应积分

退款记录列表：
  · 独立列表，展示所有退款记录及状态

【积分流水总表】

  · 平台所有用户的积分变动记录汇总
  · 筛选：变动类型（充值/注册赠送/邀请奖励/生成消耗/退款补偿/管理员手动调整）
  · 支持导出 Excel


────────────────────────────────────────
3.6 收入与财务统计
────────────────────────────────────────

收入总览：
  · 今日 / 本周 / 本月 / 本年收入金额
  · 与上一周期对比（增长/下降百分比）
  · 月度收入折线图
  · 收入来源分布（按套餐档位占比饼图）

套餐销量统计：
  · 各套餐累计销售数量 + 销售额
  · 各套餐占总销售额百分比
  · 时间维度筛选

退款统计：
  · 退款总金额 / 退款次数 / 退款率
  · 退款原因分布

数据导出：
  · 导出选定时间范围内的订单明细 Excel
  · 导出选定时间范围内的收入汇总报表


────────────────────────────────────────
3.7 风格与模板管理
────────────────────────────────────────

风格列表：
  · 当前所有可用风格的列表
  · 字段：风格ID / 风格名称 / 状态（上线/下线/开发中）/ 创建时间 / 关联模板数量

风格管理操作：
  · 新增风格（填写风格名称、描述、适用场景标签）
  · 上线/下线某风格（下线后前台用户不可选用）
  · 编辑风格名称和描述

PPT 页面模板库管理：

  每个风格下包含多种页面类型的模板，管理员可管理这些模板：

  模板列表：
    · 筛选：所属风格 / 页面类型（封面/目录/内容页/图表页等）/ 状态
    · 列表字段：模板ID / 缩略图预览 / 模板名称 / 所属风格 / 页面类型 / 状态 /
                上传时间 / 使用次数

  模板操作：
    · 上传新模板（上传 .pptx 源文件 + 缩略图 + 填写基本信息）
    · 编辑模板信息（名称、所属风格、页面类型标签）
    · 启用/禁用模板（禁用后 AI 生成时不会选用该模板）
    · 删除模板（删除前检查是否有生成记录引用，有则提示）
    · 预览模板（在线预览模板效果）

  模板质量评分：
    · 记录该模板被 AI 选用的次数和用户满意度（如果后续做用户评分功能）


────────────────────────────────────────
3.8 风格库内容管理
────────────────────────────────────────

前台展示的「风格库」中，案例内容由管理员在后台维护。

案例列表：
  · 字段：案例ID / 案例名称 / 所属分类 / 封面缩略图 / 包含页数 /
          状态（展示中/已隐藏）/ 创建时间 / 展示顺序

案例管理操作：
  · 新增案例（上传各页截图图片 + 填写名称 + 选择所属分类 + 设置排序权重）
  · 编辑案例（修改名称、分类、排序权重、替换图片）
  · 隐藏/展示切换
  · 删除案例

分类管理：
  · 风格分类列表：分类ID / 分类名称 / 排序 / 状态
  · 新增分类 / 编辑名称 / 调整排序 / 启用禁用

排序规则：
  · 支持拖拽排序案例展示顺序
  · 支持置顶某案例


────────────────────────────────────────
3.9 内容安全与违规管理
────────────────────────────────────────

敏感词库管理：
  · 敏感词列表（可添加/删除）
  · 分类：政治类 / 违法类 / 广告类 / 自定义类
  · 用户输入内容在生成前经过敏感词过滤

违规记录列表：
  · 触发时间 / 用户 / 触发词 / 原始输入内容（截断脱敏）/ 处理结果
  · 处理状态：自动拦截已处理 / 待人工审核 / 人工已处理

人工审核队列：
  · 需要人工判断的内容（如模糊词、误判白名单请求）
  · 审核操作：通过（补发积分）/ 拒绝（维持拦截，可选是否封号）

用户举报管理（二期）：
  · 用户举报其他用户生成内容的举报记录
  · 处理状态追踪


────────────────────────────────────────
3.10 邀请活动管理
────────────────────────────────────────

邀请规则配置：
  · 邀请人奖励积分数量（可配置）
  · 被邀请人首次注册奖励积分数量（可配置）
  · 邀请活动开关（开启/关闭整个邀请奖励机制）
  · 单账号最多可获得邀请奖励次数上限（防刷）

邀请数据统计：
  · 总邀请次数
  · 今日新邀请数
  · 通过邀请注册的用户总数
  · 因邀请发放的积分总量
  · 邀请转化率（邀请人/被邀请人首次生成比）

邀请排行榜：
  · 邀请数量最多的用户 Top 50 列表
  · 字段：用户昵称 / 邀请成功数 / 获得奖励积分 / 最近邀请时间

异常邀请监控：
  · 短时间内同一 IP 大量注册（疑似刷邀请）的用户列表
  · 管理员可标记为异常并撤销对应积分


────────────────────────────────────────
3.11 站内通知与消息管理
────────────────────────────────────────

系统公告：
  · 新增公告：标题 + 正文 + 生效时间 + 过期时间 + 是否弹窗展示
  · 公告列表：状态（展示中/已过期/草稿）+ 创建时间
  · 编辑/删除/立即下线

顶部横幅通知（对应前台导航栏滚动公告）：
  · 管理横幅文案内容（可多条轮播）
  · 支持添加链接（点击跳转）
  · 可设置生效时间段

给指定用户发送站内消息：
  · 输入用户ID/手机号 + 消息内容
  · 支持批量发送（上传用户ID列表 + 消息内容）

消息发送记录：
  · 时间 / 发送类型（系统公告/单独消息/批量消息）/ 接收用户数 / 内容摘要


────────────────────────────────────────
3.12 AI 生成配置管理
────────────────────────────────────────

AI 调用参数配置（核心配置，修改需超级管理员权限）：

  模型选择配置：
    · 当前使用的 AI 模型版本（可切换）
    · 各风格对应使用的模型（高质量模型/快速模型）

  生成质量参数：
    · 单页美化：默认生成参数（温度值等 AI 参数）
    · 成套 PPT：默认生成参数

  积分消耗配置：
    · 各操作对应的积分消耗数量（可在此直接修改，前台实时生效）
    · 修改记录：操作人 / 修改时间 / 修改前后数值

  并发控制：
    · 最大同时生成任务数（超出后进入排队）
    · 单用户同时最多允许几个任务在排队中

  系统提示词（Prompt）管理：
    · 单页美化的系统 Prompt 模板（可在后台直接修改）
    · 成套 PPT 大纲生成的 Prompt 模板
    · 成套 PPT 内容生成的 Prompt 模板
    · 每个 Prompt 保存历史版本，支持一键回滚
    · 修改前需填写修改说明

  风格 Prompt 管理：
    · 每个风格（如创赛精美）有独立的风格描述 Prompt
    · 可在此管理和调整

  失败重试策略：
    · AI 调用失败后自动重试次数（0-3次）
    · 重试间隔（秒）

3.13 积分套餐与价格管理
────────────────────────────────────────

套餐列表：
  · 字段：套餐ID / 套餐名称 / 积分数量 / 售价（元）/
          是否推荐 / 状态（上线/下线）/ 创建时间

套餐管理操作：
  · 新增套餐（填写名称 + 积分数量 + 价格 + 是否推荐 + 展示顺序）
  · 编辑套餐（名称、价格等，已购买用户不受影响）
  · 上线/下线套餐（下线后前台不展示，不可购买）
  · 删除套餐（仅可删除未有订单的套餐）

注册赠送积分配置：
  · 新用户注册赠送积分数量（可修改，实时生效）

邀请奖励积分配置：
  · 与「邀请活动管理」模块联动


────────────────────────────────────────
3.14 操作日志
────────────────────────────────────────

记录所有后台管理员的操作行为，不可删除，永久保留：

  · 时间 / 操作管理员 / 操作模块 / 操作类型 / 操作对象（ID）/ 操作详情 / 操作IP

筛选：
  · 操作管理员 / 操作时间范围 / 操作模块 / 操作类型

用途：问题追溯、权限审计、防止内部数据篡改


────────────────────────────────────────
3.15 系统配置
────────────────────────────────────────

基础配置（超级管理员权限）：
  · 网站名称 / 网站 Logo（可上传）
  · 前台各页面 SEO 标题 / 描述
  · 备案号（前台底部显示）
  · 客服联系方式（微信号 / 电话 / 邮箱）
  · 用户协议内容（富文本编辑）
  · 隐私政策内容（富文本编辑）

功能开关：
  · 新用户注册开关（关闭后前台不可注册）
  · 积分充值功能开关（关闭后前台不可充值，用于紧急维护）
  · 成套 PPT 生成功能开关
  · 单页 PPT 生成功能开关
  · 邀请功能开关

维护公告：
  · 开启系统维护模式（前台显示维护中页面，后台管理员可正常访问）
  · 预计恢复时间配置
